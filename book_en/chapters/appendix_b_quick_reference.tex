\texttt{xlags} = lags of each regressor; includes contemporaneous $x$ automatically.}

\section{Building a panel (formulas)}

To build a panel from a stacked DataFrame ($N$ units, $T$ periods):

\begin{lstlisting}[style=inline]
generate df unit = (_n - 1) % N + 1
generate df t    = (_n - (_n - 1) % N - 1) / N + 1
\end{lstlisting}

Where \texttt{N} is the number of units. Replace \texttt{N} with the numeric value.
