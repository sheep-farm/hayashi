\section{The Zellner Estimator}

When estimating a system of $M$ equations with the same (or different)
regressors and errors are correlated across equations, equation-by-equation
OLS is inefficient. Zellner's (1962) \textbf{SUR} (\emph{Seemingly Unrelated
Regressions}) estimator exploits this correlation via joint GLS.

System of $M$ equations:
\[
  y_m = X_m\,\beta_m + \varepsilon_m, \quad m = 1,\ldots,M
\]

Stacking: $Y = X\beta + \varepsilon$, with $E[\varepsilon\varepsilon'] = \Sigma \otimes I_n$,
where $\Sigma$ is the cross-equation covariance matrix ($M \times M$).

The joint GLS estimator:
\[
  \hat\beta_{\mathrm{SUR}} = (X'(\Sigma^{-1}\otimes I)X)^{-1}\,X'(\Sigma^{-1}\otimes I)\,Y
\]

\textbf{Efficiency gain}: SUR $\geq$ OLS when $\Sigma$ has non-zero off-diagonal
elements — i.e., when equation errors are correlated. If $\Sigma$ is diagonal
or if the regressors are identical, SUR $=$ OLS.

\section{DGP Verification}

DGP with two strongly correlated errors ($\rho = 0{,}7$):

\[
  y_1 = 1 + 2\,x_1 + \varepsilon_1 \qquad y_2 = 0{,}5 + 1{,}5\,x_2 + \varepsilon_2
\]
\[
  \mathrm{Cov}(\varepsilon_1, \varepsilon_2) = 0{,}7\,\sigma_1\,\sigma_2
\]

\begin{lstlisting}[caption={SUR DGP and comparison with equation-by-equation OLS}]
seed(42)
generate df x1 = rnormal()
generate df x2 = rnormal()
generate df e1 = (rnormal() - 0.5) * 1.7321
generate df e2 = 0.7 * e1 + 0.7141 * (rnormal() - 0.5) * 1.7321
generate df y1 = 1.0 + 2.0 * x1 + e1
generate df y2 = 0.5 + 1.5 * x2 + e2

let m_ols1 = ols(y1 ~ x1, df)
let m_ols2 = ols(y2 ~ x2, df)
esttab(m_ols1, m_ols2)

let m_sur = sur(df, y1 ~ x1, y2 ~ x2)
display m_sur
\end{lstlisting}

% ─────────────────────────────────────────────────────────────────────────────
\section{Syntax}

\begin{lstlisting}
sur(df, y1 ~ x1 + x2, y2 ~ x3 + x4)
sur(df, y1 ~ x1, y2 ~ x2, y3 ~ x3)        // 3 equations
\end{lstlisting}

The first argument is the DataFrame; the remaining arguments are formulas (one per equation).

\begin{nota}
  If all equations have the same regressors, SUR = OLS and there is no
  efficiency gain. The gain is maximized when the regressors are orthogonal
  across equations and the cross-equation error correlation is high.
\end{nota}
