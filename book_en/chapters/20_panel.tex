\section{The model}

A panel observes $n$ individuals over $T$ periods. The general model is:
\[
  y_{it} = \alpha_i + X_{it}\beta + \varepsilon_{it},
  \qquad i = 1,\ldots,n;\quad t = 1,\ldots,T
\]

where $\alpha_i$ is the \textbf{unobserved individual effect} --- time-fixed
characteristics that differ across individuals (ability, culture, location).
Cross-sectional or pooled OLS ignores $\alpha_i$, treating it as part of
the error:

\[
  \mathrm{plim}\;\hat\beta_{\mathrm{OLS}}
  = \beta + \frac{\mathrm{Cov}(x_{it},\,\alpha_i)}{\mathrm{Var}(x_{it})}
\]

The bias disappears only if $\mathrm{Cov}(x_{it}, \alpha_i) = 0$ --- a strong
assumption in most applications.

% ─────────────────────────────────────────────────────────────────────────────
\section{Fixed Effects}

The \textbf{within} estimator eliminates $\alpha_i$ by subtracting each
individual's time mean:
\[
  \tilde{y}_{it} = y_{it} - \bar{y}_i, \qquad
  \tilde{X}_{it} = X_{it} - \bar{X}_i
\]

The Fixed Effects estimator is then OLS on the demeaned variables:
\[
  \boxed{\hat\beta_{\mathrm{FE}} = (\tilde{X}'\tilde{X})^{-1}\tilde{X}'\tilde{y}}
\]

Because the within transformation annihilates $\alpha_i$ for any value of
$\mathrm{Cov}(x_{it}, \alpha_i)$, the estimator is consistent even under
endogeneity of the individual effect. The variance is:
\[
  \widehat{\mathrm{Var}}[\hat\beta_{\mathrm{FE}}] = s^2_{\mathrm{FE}}(\tilde{X}'\tilde{X})^{-1},
  \qquad
  s^2_{\mathrm{FE}} = \frac{\hat\varepsilon'\hat\varepsilon}{nT - n - k}
\]

\begin{aviso}
  Variables that do not vary over time (sex, race, region of birth) are
  eliminated by the within transformation. FE cannot estimate their effect.
  Use RE or hybrid estimators in those cases.
\end{aviso}

% ─────────────────────────────────────────────────────────────────────────────
\section{Random Effects}

If $\alpha_i \sim \mathrm{IID}(0, \sigma^2_\alpha)$ and
$\mathrm{Cov}(x_{it}, \alpha_i) = 0$, then $\alpha_i$ is part of the composite error
$v_{it} = \alpha_i + \varepsilon_{it}$. The error structure implies GLS with
quasi-demeaning:
\[
  y^*_{it} = y_{it} - \theta\bar{y}_i,
  \qquad
  \theta = 1 - \frac{\sigma_\varepsilon}{\sqrt{\sigma^2_\varepsilon + T\sigma^2_\alpha}}
  \in [0,1)
\]

The RE estimator is OLS on $y^*_{it}$ and $X^*_{it}$. It is more efficient than
FE under $H_0: \mathrm{Cov}(x_{it},\alpha_i) = 0$, but \emph{inconsistent}
when that assumption fails.

% ─────────────────────────────────────────────────────────────────────────────
\section{Hausman Test}

The Hausman test compares FE (always consistent) with RE (consistent under
$H_0$, efficient):
\[
  H_0: \mathrm{Cov}(x_{it}, \alpha_i) = 0
  \quad\Longrightarrow\quad
  \mathrm{plim}(\hat\beta_{\mathrm{FE}} - \hat\beta_{\mathrm{RE}}) = 0
\]

The test statistic is:
\[
  H = (\hat\beta_{\mathrm{FE}} - \hat\beta_{\mathrm{RE}})'
      \bigl[\widehat{\mathrm{Var}}(\hat\beta_{\mathrm{FE}}) -
             \widehat{\mathrm{Var}}(\hat\beta_{\mathrm{RE}})\bigr]^{-1}
      (\hat\beta_{\mathrm{FE}} - \hat\beta_{\mathrm{RE}})
  \;\xrightarrow{d}\; \chi^2(k)
\]

Rejecting $H_0$ indicates that RE is inconsistent --- use FE.

\subsection{Robust Hausman}

The classical Hausman test requires the RE estimator to be efficient
under $H_0$. Under heteroskedasticity or clustering, this fails and the
test statistic can be negative or misleading. The robust version
(Cameron \& Trivedi 2005; Wooldridge 2010) uses the robust covariance
of the coefficient differences:

\[
  H_{\mathrm{robust}} = (\hat\beta_{\mathrm{FE}} - \hat\beta_{\mathrm{RE}})'
      \bigl[\widehat{\mathrm{Var}}_{\mathrm{robust}}(\hat\beta_{\mathrm{FE}} -
             \hat\beta_{\mathrm{RE}})\bigr]^{-1}
      (\hat\beta_{\mathrm{FE}} - \hat\beta_{\mathrm{RE}})
  \;\xrightarrow{d}\; \chi^2(k)
\]

\begin{lstlisting}[caption={Robust Hausman test}]
hausman_robust(m_fe, m_re)
\end{lstlisting}

Alias: \texttt{hausman\_r}.

\section{DGP Verification}

\subsection*{Exact recovery (no noise)}

The DGP uses 100 individuals $\times$ 10 periods ($n = 1000$). The fixed effect
$\alpha_i = 0{,}05\cdot i$ is deterministic and correlated with $x$:
\[
  x_{it} = \alpha_i + u_{it},\quad u \sim \mathcal{N}(0,1)
  \qquad
  y_{it} = 1{,}0 + 2{,}0\,x_{it} + \alpha_i \quad(\varepsilon = 0)
\]

Without idiosyncratic noise the within transformation eliminates $\alpha_i$
exactly, and FE recovers $\beta$ to machine precision:

\begin{lstlisting}[caption={FE with correlated fixed effect --- no noise}]
seed(42)
input df
id
1
end
for i in 1..=10 { df = append(df, df) }
df |> filter(_n <= 1000)
generate df id    = (_n - 1) % 100 + 1
generate df alpha = id * 0.05
generate df u     = rnormal()
generate df x     = alpha + u
generate df y     = 1.0 + 2.0*x + alpha
let m_ols = ols(y ~ x, df)
let m_fe  = fe(y ~ x, df, id=id)
esttab(m_ols, m_fe)
\end{lstlisting}

\subsection*{Hausman with noise}

With idiosyncratic noise $\varepsilon \sim \mathcal{N}(0,1)$, FE remains
consistent and RE is inconsistent (since $\alpha_i$ is still correlated
with $x$). Hausman should reject $H_0$:

\begin{lstlisting}[caption={Hausman --- FE vs RE with correlated effect}]
// same DGP; adds idiosyncratic noise
generate df e  = rnormal()
generate df y2 = 1.0 + 2.0*x + alpha + e
let m_fe2 = fe(y2 ~ x, df, id=id)
let m_re2 = re(y2 ~ x, df, id=id)
esttab(m_fe2, m_re2)
hausman(m_fe2, m_re2)
\end{lstlisting}

\section{Syntax}

\begin{lstlisting}
fe(formula, df, id=col)
re(formula, df, id=col)
hausman(fe_model, re_model)
\end{lstlisting}

The \lstinline{id} column identifies the individual. FE automatically removes
the intercept (absorbed by the individual effects).

\begin{lstlisting}[caption={Complete workflow}]
load "panel.csv" as df

let m_fe = fe(wage ~ educ + exp, df, id=id)
let m_re = re(wage ~ educ + exp, df, id=id)

esttab(m_fe, m_re)
hausman(m_fe, m_re)
\end{lstlisting}

\section{Standard error options}

\begin{lstlisting}
fe(y ~ x, df, id=id)                        // classical
fe(y ~ x, df, id=id, cov=robust)            // HC1
// cluster by individual
fe(y ~ x, df, id=id, cov=cluster, cluster=id)
\end{lstlisting}

\begin{nota}
  In panels, errors clustered by individual (\lstinline{cluster=id})
  are the recommended default: they correct heteroscedasticity and serial
  correlation within the individual simultaneously.
\end{nota}

\begin{lstlisting}
let m = fe(y ~ x, df, id=id)

\end{lstlisting}

\section{Panel diagnostics}

The choice between pooled OLS, Fixed Effects, and Random Effects depends on
formal tests. Hayashi implements all standard tests from the literature.

\subsection{Pooled OLS vs.\ Random Effects}

\textbf{Breusch-Pagan LM} (\texttt{bptest}): tests $H_0: \sigma_u^2 = 0$
(no panel effect --- pooled OLS is adequate):

\begin{lstlisting}
bptest(df, y ~ x, id="entity")
\end{lstlisting}

If $H_0$ is rejected, use RE or FE instead of pooled OLS.

\subsection{Pooled OLS vs.\ Fixed Effects}

\textbf{F-test for FE} (\texttt{ftest\_fe}): tests $H_0$: all individual
effects are zero:

\begin{lstlisting}
ftest_fe(df, y ~ x, id="entity")
\end{lstlisting}

Rejecting $H_0$ indicates FE is preferable to pooled OLS.

\subsection{Fixed Effects vs.\ Random Effects}

\textbf{Hausman} (\texttt{hausman}): tests $H_0$: RE is consistent
(individual effects uncorrelated with regressors):

\begin{lstlisting}
hausman(m_fe, m_re)
\end{lstlisting}

Rejecting $H_0$ implies FE (consistent under correlation); not rejecting
implies RE (more efficient).

\textbf{Mundlak} (\texttt{mundlak}): a generalization of Hausman that
augments the model with entity means of regressors:

\begin{lstlisting}
mundlak(df, y ~ x1 + x2, id="entity")
\end{lstlisting}

\subsection{Serial correlation and cross-sectional dependence}

\textbf{Wooldridge} (\texttt{wooldridge}): tests $H_0$: no first-order
serial correlation in idiosyncratic errors:

\begin{lstlisting}
wooldridge(df, y ~ x, id="entity", time="time")
\end{lstlisting}

\textbf{Pesaran CD} (\texttt{pesaran}): tests $H_0$: no cross-sectional
dependence. Relevant for large-$N$ panels where spatial or network spillovers
may exist:

\begin{lstlisting}
pesaran(df, y ~ x, id="entity", time="time")
\end{lstlisting}

\subsection{Arellano-Bond m1/m2}

\textbf{Arellano-Bond} (\texttt{abtest}): tests serial correlation in
first-differenced residuals. $m_1$ should reject $H_0$ (FD induces AR(1) by
construction); $m_2$ should not reject $H_0$ (validates the $y_{i,t-2}$
instruments for GMM):

\begin{lstlisting}
abtest(df, y ~ x, id="entity", time="time")
\end{lstlisting}
