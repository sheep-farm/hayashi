# Elastic Net regression on Wooldridge `hprice1`

This validation case estimates an Elastic Net regression of log house price on log lot size, log square footage, number of bedrooms and a colonial dummy using the Wooldridge `hprice1` dataset.

## Model

```
lprice ~ llotsize + lsqrft + bdrms + colonial
```

## Dataset

- **Name:** `wooldridge::hprice1`
- **Source:** R package `wooldridge`; also available in Python via `wooldridge`.
- **Licence:** Public teaching dataset.
- **Size:** 88 observations × 11 variables (log-transformed variables generated by the reference scripts).

## Reference implementation

- **Python:** primary reference implementation.
- **R:** a reference script is provided but currently fails because the `glmnet` and `jsonlite` packages are not installed. The validation currently relies on the Python reference.
- **Hayashi:** `elasticnet(lprice ~ llotsize + lsqrft + bdrms + colonial, df, alpha=0.1, l1_ratio=0.5)`

## Compared quantities

- coefficients only (Elastic Net has no analytical standard errors in these implementations)

## Tolerances

| Quantity | Tolerance | Rationale |
|---|---|---|
| coefficients | 1e-1 | Different solvers and internal standardisation may produce small differences; the Python reference uses the same alpha/l1_ratio parameterisation as Hayashi |
