\section{Limitações dos estimadores estáticos}

O cap.~\ref{cap:painel} apresentou FE e RE para painéis com efeito individual
constante. Três extensões importantes surgem em aplicações empíricas:

\begin{enumerate}
  \item \textbf{Dinâmica}: quando $y_{i,t-1}$ é regressor, FE é viesado
  (\emph{viés de Nickell}): $\hat\rho_{\mathrm{FE}} \xrightarrow{p} \rho - O(1/T)$.
  O estimador Arellano-Bond usa GMM em primeiras diferenças para contornar.
  \item \textbf{Dependência cross-seccional}: quando erros de diferentes
  unidades são correlacionados (dados macroeconômicos de países), os SEs
  OLS e FE são inválidos. PCSE (Beck e Katz 1995) corrige.
  \item \textbf{Dados não-gaussianos em painel}: GEE (Liang e Zeger 1986)
  estende modelos de margem para dados de painel com famílias exponenciais.
\end{enumerate}

\subsection*{Arellano-Bond (AB)}

O estimador diff-GMM de Arellano e Bond (1991) toma primeiras diferenças
para eliminar o efeito individual:

\[
  \Delta y_{it} = \rho\,\Delta y_{i,t-1} + \beta\,\Delta x_{it} + \Delta\varepsilon_{it}
\]

Instrumentos: $y_{i,t-2}, y_{i,t-3}, \ldots$ são válidos para $\Delta y_{i,t-1}$
pois são não correlacionados com $\Delta\varepsilon_{it}$, contanto que os
erros em nível não sejam serialmente correlacionados de ordem 2 (teste $m_2$).

\section{Verificação por DGP}

\begin{lstlisting}[caption={DGP de painel com endogeneidade via efeito individual}]
seed(42)
generate df unit  = (_n - 1) % 50 + 1
generate df t     = (_n - (_n-1) % 50 - 1) / 50 + 1
generate df alpha = unit * 0.05        // efeito individual correlacionado com x
generate df x     = alpha + (rnormal() - 0.5) * 1.7321
generate df e     = (rnormal() - 0.5) * 1.7321
generate df y     = 1.0 + 1.0 * x + alpha + e  // verdadeiro: b=1.0

// OLS viesado (mistura alpha_i com x)
let m_ols  = ols(y ~ x, df)

// FE consistente (elimina alpha_i)
let m_fe   = fe(y ~ x, df, id=unit)

// AB-GMM (dinâmico com lags como instrumentos)
let m_ab   = ab(y ~ x, df, id=unit, time=t, lags=2)

// PCSE
let m_pcse = pcse(y ~ x, df, id=unit, time=t)

// GEE
let m_gee  = gee(y ~ x, df, id=unit)
\end{lstlisting}
